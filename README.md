# My Expense

Personal finance & agent cash management app.

## Features

- **Dashboard** — Balance overview, monthly budget progress, income vs expense, recent transactions
- **Add Transactions** — Income/expense entries with categories, plus agent cash management mode (cash-in, cash-out, transfer, bill payment, top-up) with customer & commission tracking
- **Analytics** — Bar chart by category, pie chart breakdown, income/expense stats
- **Transaction History** — Searchable, filterable (All/Income/Expense) full list
- **Manage Wallets** — Add/remove wallet sources (KPay, WavePay, Cash, etc.)
- **Settings** — Edit name, currency (MMK/USD/THB/SGD), monthly budget, starting balance
- **Export Data** — Generate PDF with transaction table and summary, share via system share sheet
- **Onboarding** — 4-step wizard for name, currency, wallets, and starting balance
